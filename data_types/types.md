Basic Data Types 

*Boolean
 -true, false
*Integer
 -1, 2, 50, 99, -2
*Double / Float
 -1.1, 5.5, 200.0001, 2.0
*Character
 -'A', 'B', 'c', '6', '$'
 notes: to represent characters we should use single quotes
*String
 -"Helo", "string", "this is a string", "its 42"
 notes: to represent strings we should use double quotes
